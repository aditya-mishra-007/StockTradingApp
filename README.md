# 📈 StockTradingApp

A modern stock market tracking application that allows users to monitor real-time stock prices, explore company insights, and manage personalized watchlists.

Built with modern web technologies to provide a fast, scalable, and user-friendly experience.

</div>

---

## 🚀 Features

- 📊 Real-time stock price tracking
- ⭐ Watchlist management
- 🔔 Price alerts and notifications
- 📈 Interactive stock charts
- 📰 Company insights and market data
- 🤖 AI-powered summaries and analysis
- 🔐 Secure authentication system
- 📧 Email notifications

---

## 🛠 Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Shadcn UI**
- **MongoDB**
- **Better Auth**
- **Inngest**
- **Finnhub API**
- **Nodemailer**
